INTRODUCTION
------------

The Warpwire Mediacore Filter module allows for the translation of Mediacore embed 
urls into Warpwire embedded content.

An example of an embedded Mediacore asset is as follows:

[mediacore:https://example.mediacore.com/media/sample-media-file/]

REQUIREMENTS
------------

Modules
  - Warpwire - The Warpwire module


INSTALLATION
------------

The Warpwire Mediacore Filter module can be installed via the standard Drupal installation process
(http://drupal.org/node/895232).  Alternatively, simply extract the Warpwire module into
/sites/all/modules/ - the resulting folder will be /sites/all/modules/warpwire_mediacore.
